¿Qué es CMD?

El CMD (Símbolo del sistema) es una herramienta de línea de comandos que se utiliza en sistemas operativos Windows. Permite a los usuarios ejecutar programas y comandos, gestionar archivos y realizar tareas administrativas sin necesidad de una interfaz gráfica.

Comandos Básicos de CMD

Comando Descripción
dir Muestra una lista de archivos y carpetas en el directorio actual.
cd Cambia el directorio actual a otro especificado.
copy Copia archivos de una ubicación a otra.
del Elimina archivos del directorio actual.

Ejemplo de Uso de CMD

Un ejemplo sencillo de uso de CMD es cambiar al directorio "Documentos" y listar los archivos que contiene:

cd Documentos
dir

En este caso, el primer comando cambia el directorio actual a "Documentos", y el segundo comando lista todos los archivos en ese directorio.
